# 聊天系统

## 目标

涉及的功能模块如下：

- [x] 利用TCP进行消息的传递  

- [ ] 利用UDP进行消息的传递 

- [ ] 利用TCP进行文件的传输

- [ ] 编写一个用户登录系统。使用简单SQLite数据库。

- 其他目标  
美化界面。尽量与QQ等界面相同。
